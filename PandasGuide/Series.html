What is a series?

Introduction

When we grab a column of a data frame with dataframe['column'], we are left with a data structure called a series. What is it? What can we do with it?

Structure

A series is a kind of indexed list. If you got the series from a dataframe, the series inherits its indexing (recall that the indexing gives the names to the entries). There are, therefore, two ways of referencing a particular value in the series.

  1. series['a'] - Use the index to specify the value you need
  2. series.hloc[n]
    3. - Gets the nth entry of the series by counting from the first being the 0th.
While possible if the indexing doesn't include numbers, it is bad practice to use series[n] to indicate that you want the n+1th entry of the series.

Creating a series from scratch

You do this with the function pd.Series(data =). Your data can either be a dictionary, such that the keys form the indices and the values the entries, or a list, in which case you introduce the indices via the parameter index = [].

Vectorization

Series are vectorized. Loosely speaking, this means that if you perform an operation on a series that is meant for its entries, Pandas knows to perform the operation on its entries after all. The following examples will hopefully be illustrative. 


                        series = pd.Series{data = [0,1,2], index = [0,1,2]}
                        series = series + 1
                        print(series)

Output: 


                        0    1
                        1    2
                        2    3

1 is added to each entry of the series. 


                        series = pd.Series{data = [0,1,2], index = [0,1,2]}
                        bool = (series == 0)
                        print(bool)

Output: 


                        0    True
                        1    False
                        2    False

Each entry of series was judged whether it equaled 0 and the series of judgments was saved to bool. 


                        series1 = pd.Series({'a':0,'b':1,'c':2})
                        series2 = pd.Series({'a':0,'b':1,'d':2})
                        print(series1+series2)

Output: 


                        a    0.0
                        b    2.0
                        c    NaN
                        d    NaN

Pandas tries to match up correspondingly indexed entries in the two series and add each pair together. Note that when one of the series is missing a correspondingly indexed entry, the result for that index in the resultant series is NaN!
