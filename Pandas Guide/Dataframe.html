What is a dataframe?

Note that the following coverage is not comprehensive!

Introduction

A dataframe is the primary data structure of Pandas. It is a kind of two dimensional array where the rows refer to objects and the columns to attributes.

Making a dataframe

You have some choices for how you want to make a dataframe:

  1. Making one from scratch.
  2. Importing a JSON, CSV, XSL, etc. file and converting it into a dataframe.

Making a dataframe from scratch

Making a dataframe using a dictionary

You can use a dictionary to make a dictionary, so long as its values are lists of equal length. Each key will label a particular column (name an attribute), and the key's corresponding value will be a list of entries for the column. Corresponding value entries correspond to the same object.

Call pd.DataFrame(data = dictionary) to convert your dictionary into a dataframe. 


                        dictionary = {'1': ['a_1','b_1'],'2': ['a_2','b_2']}
                        df = pd.DataFrame(data = dictionary)
                        print(df)

Output:


                            1   2
                        0   a_1 a_2
                        1   b_1 b_2

The column on the left is called the index for the rows. They are the names of your objects. If the source of your data did not specify names, then they will be auto-generated starting from 0, as in the above example. If you would prefer to set your own names, you can use the parameter index = [insert, names, here] when calling pd.DataFrame.

Making a dataframe using a numpy array

If you have a two dimensional numpy array, you can also turn that into a dataframe using pd.DataFrame(). In this case, a row will correspond to the values of an object. You can add attribute names using the parameter columns = [insert, names, here]. See the following example: 


                        array = np.array([['a_1','a_2'],['b_1','b_2']])
                        array_df = pd.DataFrame(data = array, columns= ['1', '2'])
                        print(array_df)

Output:


                            1    2
                        0  a_1  a_2
                        1  b_1  b_2

Converting a file into a dataframe

This is easy to do for Excel (.xls), JSON (.json), and CSV (.csv) files. Simply use one of the following, repalcing data with the path of the file in question: 


                        df_excel = pd.read_excel('data.xlsx')
                        df_json = pd.read_json('data.json')
                        df_csv = pd.read_csv('data.csv')

Note that JSON files are basically dictionaries, so that they allow for a lot more complexity than what is allowed in a dataframe. Make sure that your file is formatted appropriately for dataframe-ification!

Getting a row or a column of a dataframe

To get a row of a dataframe, you either reference it by its index (dataframe.loc['index']) or you use the .iloc[n] method to reference it by its position (count from the top, starting at 0).
