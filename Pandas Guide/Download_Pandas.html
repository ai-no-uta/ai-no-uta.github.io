Downloading Pandas in Windows

Prerequisites

Note that this is a guide for Windows 10 specifically. First, make sure that you have pip installed. According to Wikipedia, pip should come preinstalled with version 2.7.9 and later for python2 and version 3.4 and newer for python3.

To check the version of Python you have, input python --version into Command Prompt. To check whether pip is installed, input pip --version. If pip is not installed, then Command Prompt will display an error along the lines of is not recognized as an internal or external command, operable program or batch file.

If pip is not installed, then rectifying that is your first line of business. Save this page as a .py file onto your computer and run py C:\your_path_here.py on Command Prompt.

Installing Pandas

Check whether Pandas is already on your computer by running pip show pandas. If it's installed, you will see a bunch of information about it, including the version number, a summary about what it does, and a link to its official website.

If it isn't installed, you will see WARNING: Package(s) not found: pandas.

In this case, run pip install pandas into Command Prompt, and you should be good to go!

Running Pandas

Typically, you import Pandas with the line import pandas as pd.
