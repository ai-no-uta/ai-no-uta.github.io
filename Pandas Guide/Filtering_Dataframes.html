Filtering a Dataframe

Introduction

Sometimes, we want to only get a part of a dataframe. We want to only look at rows with particular values. How do we do that?

Filtering rows by values

Suppose that we only want to look at the rows of a dataframe with the value n for column col. To do so, we input something like: 


                        newDataframe = Dataframe[Dataframe['col'] == n]]

Let us break down what the code is doing. Dataframe['col'] nabs the column 'col' as a data structure called a series. Series are nice because they are vectorized: when you try to perform an operation on it, the operation will actually be performed on each of the entries, and the output will be a series of the same length consisting of all of the results of the entrywise operations. In this case, the result is a series of Boolean values where (Dataframe['col'] == n).iloc[k] = True if Dataframe['col'].iloc[k] == n and (Dataframe['col'] == n).iloc[k] = False otherwise.

Given a boolean series a with the same number of entries as the number of objects described by the dataframe dataframe, dataframe[a] returns a dataframe containing the rows whose index corresponds to entries of a that are True.

Putting these two together, we see how we have filtered our dataframe.
